 一个粗糙的“批量文件重命名工具”，后面有时间慢慢优化吧！

 灵感来自于[https://github.com/guiqiqi/AllRename](https://github.com/guiqiqi/AllRename)，感谢原作者~。但是这个没有递归查找目录下文件的功能，只能单个文件夹的操作，所以想着实现一下，用着能方便一点。

 本项目使用了`Vue + TS + electron`开发。
